## 导入导出规范检查清单

### 1. 导入顺序
- [ ] React 导入在最前
- [ ] 第三方库其次
- [ ] `@/` 内部模块第三
- [ ] 相对路径导入第四
- [ ] 样式文件最后

### 2. 类型导入
- [ ] 使用 `import type` 导入类型
- [ ] 类型导入与值导入分离

### 3. 导出规范
- [ ] 页面组件使用默认导出
- [ ] 工具函数使用命名导出
- [ ] 组件目录使用 `index.ts` barrel 导出

### 4. 命名规范
- [ ] 导入名与导出名一致
- [ ] 避免命名冲突
- [ ] 重命名时语义清晰

### 5. 路径规范
- [ ] 优先使用绝对路径 `@/`
- [ ] 相对路径只用于同级目录
- [ ] 避免深层相对路径 `../../../`
